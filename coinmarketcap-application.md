## Project Description

BrainArk is a custom Ethereum-compatible blockchain built on Hyperledger Besu, featuring:

- **IBFT Consensus:** Istanbul Byzantine Fault Tolerance for fast, secure transactions
- **4-Node Network:** Decentralized validator network ensuring reliability
- **Low Gas Fees:** Optimized for cost-effective transactions (1000 wei gas price)
- **EVM Compatible:** Full Ethereum Virtual Machine compatibility
- **Native Coin:** BAK serves as the native cryptocurrency for all network operations

## Use Cases

1. **Transaction Fees:** BAK is used to pay for all network transactions (gas fees)
2. **Validator Rewards:** Network validators earn BAK for securing the blockchain
3. **Network Security:** BAK secures the Layer 1 blockchain through validator staking
4. **DeFi Ecosystem:** BAK powers decentralized finance applications on the network
5. **Medium of Exchange:** BAK serves as the primary currency for the BrainArk ecosystem
